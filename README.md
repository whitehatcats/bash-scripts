# Bash Scripts

A collection of Bash scripts for basic DevOps tasks and system automation.

## 📄 Scripts

### `system-info.sh`
Displays key system info such as:
- Hostname
- Uptime
- OS version
- CPU and memory stats

### `disk-check.sh`
Checks disk usage and prints a warning if it exceeds 80%.
